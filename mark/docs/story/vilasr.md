# 我的大学

## 初来咋到

大学是一个充满了想象力的地方，让我觉得一切都好像不那么真实，可能是高中的生活单调惯了。

在机场坐学校的大巴去到学校的路上，看着窗外的大太阳以及林立的建筑，还有那最喜欢的大海，脑袋空空，思考着一定要好好学习（-->在现在看来确实也没错，好好学习确实能看到很多不一样的东西）。进入大学后认识了很多的人，看到了很多的社团，也尝试着加入了社团，大学好像除了学习还有很多的班级活动，院活动，校活动，某些时候还有各个专业的活动，也有很多是强制，虽然不是很愿意（哈哈哈），但是也必须要去。

上了大学后发现时间回到了自己的手里，你可以熬夜玩电脑游戏，甚至可以半夜去海边吹吹海风亦或是凌晨去看日出，很自由。这就需要你有一个很强的时间管理能力，也看到了很多人半夜打游戏，白天睡觉，翘课（哈哈哈），同时也有需要适应的。

## 军训风云

这是我经历过最快乐的军训时光，因为我从开始笑到了结束，也间接的认识了很多的人。

为什么这么说呢！个子比较矮（哈哈哈）所以到了最后一排，呐，机会就这么来了，本身就是一个有点搞笑的人，刚好旁边的人也是一个搞笑的人，后续正前方的那位也是这类人，这不巧了吗。 在这日复一日的训练下，逐渐熟悉了起来，画风逐渐不对劲，我说普通话，他们说方言，我听不懂，说方言这两互相也听不懂，笑称加密的方言，就这样我也学起了方言。

每天最搞笑的就是练齐步了，不知道为什么我看前面那位总想笑，他看我也忍不住笑，所以每次5米距离的齐步，总是忍不住笑得发抖，然后就是大家都忍不住，因为实在是太形象了，可以说是相当的放松，方队很多，所以正式的是轮不上我们的，只需要在看台上坐着参加典礼就行。

## 一年级（上下）

这个阶段一直在尝试做很多不一样的事情，找到自己喜欢什么，好好学习数学、英语、尝试加入社团，参加一些活动，逛了学校，去了不同的学院，选修了英语演讲（上台演讲了两次，虽然很不熟练，但还是很锻炼人，特别喜欢能够上台去演讲，能够慢慢的适应不紧张）

后面，开始了图书馆管理员的工作，需要做的就是将书籍按`pn`号归位，图书馆很大，有五层，玻璃顶很好看。

大部分时间也都花在了图书馆，图书馆有电脑室，不带电脑就会去用它查资料，可以用一整天，在这里也认识了很多的人，最重要的是在这里遇到了一位高年级的同学（后面带我认识了一位嵌入式很强的大四同学，最后直博去了美国），在这里每天确实也是努力在学习，靠自己整理考试资料，印象很深刻，最后数学、物理也是接近满分（差了一两分），刚开始什么都不了解，慢慢的熟悉，尝试，了解了一些运行规律，渐渐的有了自己的目标，知道了自己要干什么。

## 二年级（上）

这是一个很大的转折点，开学的一天晚上，从图书馆回来就和一年级认识那位高年级同学去见了那位计算机很强的同学，聊了很多，也确实知道了要学好计算器这方面需要付出那些工作与努力，以及要跟对导师。之后，我便去了这位同学推荐的教授的实验室，想要加入他们，但是没有找到人，我也没泄气，刚巧我有在上那位教授的课，后面下课后，我便找到他，向他表明了我的意图，他也跟我聊了很多，下课路上人很多，刚好也是饭点，他说"我请你吃饭吧"，和他一起去了食堂（他给我刷的卡，每每想起这里都是对老师感激之情）。

当然，总不能说想加入就加入吧。自然是需要看看你有没有想学的心，而不是一时冲动。他跟我说看看你这学期的期末成绩，就这样，每天都在努力的学习，同时也没有放弃图书馆的工作，我只记得每天的很晚回去，图书馆`10:50`关门，我就`50`走哈哈哈，最重要的是我的书可以放在管理员办公室（因为我是管理员），那会不知道为什么，完全不觉得累，反而很快乐，感觉乐在其中。每天晚上一个人背着包走在校园的马路上，看着笔直且看不到头的一排排路灯，心里莫名轻松，不由得拿出手机拍照，回去后洗澡便睡觉了，但是总是难以入眠，因为总能在半夜`一点`还能听见青轴键盘的声音。第二天早上也会`八点`早起去上课或者去图书馆，晚上`11点`总喜欢拿着球去篮球场投一投，笑称释放压力（哈哈哈） ，就这样转眼到了年底，带着那份不错的成绩和忐忑的心去到了实验室，也是只有教授在，他给我泡了茶，边喝边聊，说着我的成绩不错，后面也是顺利进入了实验室，便开始了我的单片机点灯之路，这是时候也感觉自己很疯狂，每天抱着电脑学习，整理代码，运行工程....。

## 比较普通的一二年级

 这段时间逐渐适应了很多的东西，学会了如何睡好觉，如何最快的完成作业，如何去干很多自己的事情，如何合理分配自己的时间。
