# 我不是一个讲故事大师

## 经历

我一直在想怎么做好一件事，这个不难回答，就是坚持！

生活中有很多好的事情和不好的事情，教会了我很多，也变成了从小到大旁人说的懂事，但是这真的是懂事吗，我更多的是觉得无非是经历多了之后，知道了很多事情的发展是怎样的，学会了要冷静看待。

小时候对很多东西没什么概念，总是只看见眼前的美好，依稀记得玩卡带游戏的时候很开兴，冒险岛很有意思。快乐总是短暂的，高中见识很多的东西是需要努力的，并不是简单看看就会，努力总会有汇报，这大概就是对高中成绩的总结，大学一半这样吧，更多在于机会，努力要有方向。

上学时总是有目标，无论是导师的给的，还是自己定的，工作后，一时之间好像没有找到自己的目标，所以也一直再给自己找一些东西做，无论是打游戏也好，看电影也好，总感觉好像都不是自己想要的。还真是PUBG打多了想砸电脑（哈哈哈哈，太菜了）所以就想着写了这样的一个记录网站（大学就想写的，只不过一直写的不好），在Github上找的（感谢大神）

有一段时间真的很喜欢看电影，后面又看了很多大神拍的微电影，以至于有时候还去油管上找找有没有最新的，感觉他们的视频都好动感（Casey/Danny....)真的很喜欢，总算是找到了一些想要的。

平时觉得不好的时候也会去篮球馆打打篮球，以至于总是很晚去，因为没人，所以可以一个人慢慢投。搬过一次家，发现还是东西不要太多好（太难搬了）。

我发现一个问题，写东西的还是得去图书馆，环境很重要，效率会高很多。 

有时候我也有一个问题，就是很久不生病，生病就头疼 。
