# 大学逐渐有了规律

## 开始

二年级开始，逐渐学习`C++`、`Linux`、很多开发板，以及一些协议，网上找到了很多的嵌入式学习资源，自此，后面的时光都在实验室了，偶尔也会去图书馆，特别是到期末的时候，我也不知道为什么，有了总结资料后，总是能很快复习完一本书，或者老师布置的经典题型，亦或是一周刷完`B站`网课，然后莫名其妙就会了。后面成绩也不算很好，只能说勉强够用，上课的状态就这样持续到了毕业，平时学的很多，其实到三年级的时候，有很多可能都是将这些，所以能很好的理解，即使是像数据结构这样的课程，在做实验设计时，也能很快做完，理解很快。主要使平时也会去刷`leetcode`学习。

三年级，也是如此，没什么运动，但是一周也会有几天回去篮球场打打篮球，体测时也会去体育场跑跑步，至少到四年级前`1000m`都在`4`分内，依稀记得暑假都在学校学习，参加比赛。

三年级下开始，好像很多无趣的事情也变得有趣了起来，每每在中午吃饭后、下午吃饭后、晚上时候，总会拿出一个小时左右的时间和大家玩`茶杯头`，还别说，最后真让我们打通关了，不过那已经是四年级的事情了。

三年级很快，因为要忙很多的事情，够不着保研，也没考研，最后还是觉得找工作，所以三年级很多都是围绕着找工作开始的，包括去研究某些招聘网站上对应的岗位需要什么那些能力，主要做的也是根据这些来的，所以很多的时间也多在刷`力扣`，然后看很多的面试知识，对自己做的项目的深度挖掘，同时也在不断做一些新的项目，这一年，时间都在实验室，疯狂的丰富自己的知识。

暑假幸有老师推荐，去了一家公司实习，在哪很快，也学到很多的知识，感受到了很多不一样的地方，以及对项目的整体理解，需要怎么做，掌握很多规范等等，这都是学校没有详细注意到的地方，也是自己的短板，有用的知识很多，也结识了几个工作中的人，教给了我很多的道理，也让我见识到了很多的工具，后面当自己想到某些方面的时候，会很不自觉的就想到它们。

## 四年级

这个时候，其实很多不是很重要的，都在忙着投简历，刷题，笔试，面试等等。反之，课程的学习之类的并没有放在心上，这个时候，我都在图书馆，刷着`力扣`看着帖子，一坐便是一天，有笔试的时候就会找个安静的地方。

这个阶段，一度自我怀疑过，因为并不是硕士，所以很多的时候，简历其实在这里就已经被晒掉了。到最后甚至找工作的每个阶段我的经历过了，比如笔试没过，一面没过，二面没过，三面没过，有很多时候真的很怀疑自己，是不是真的可以，一度失落的很难受，甚至被打击得怀疑自己到底适不适合做程序员这份工作，但是到这学期的后半部分，我逐渐明白过来了，前面的都是什么神仙打架，优秀的人太多了。

在前期的时候有一个本地的芯片厂，笔试过了，一面也过了，二面的主管问了很多，有些东西，让我现场想了方案，感觉没把握住，最后也没过。但是这些都不是灰心的地方，每天想的就是复盘，哪里不足，就看哪方面的资料。`元旦`前后，突然接到了很多公司的面试，HR都表示对我很感兴趣，跟着推面试。在`Github`冲浪的时候，看到了一个中国科学院软件研究所（`ISCAS`）[智能软件研究中心（ISRC）](https://isrc.iscas.ac.cn/)的老师在招实习生（`PCLT`团队）,刚好看到了很多，我符合条件的岗位，我就试着投了投，没想到后面过了那个老师的测验，也过了他们团队的面试，也给我发了合同，但是这里戏剧性的一幕也来了，同一时间`小米`也给我发了offer，最后我签了三方，在这位老师这里我就给他表示我的想法，放弃了这个实习的机会。到了这里，我才终于是放松了一点，至少没有给实验室的大家拖后腿（因为同一级进lab的大部分都保研了）。
